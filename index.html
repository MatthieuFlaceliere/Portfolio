Les processus de messagerie électronique sont regroupés dans des groupes de processus. Cependant, ces groupes n'étaient pas présentés dans une IHM (Interface Homme Machine) pour l'utilisateur. Le modèle de données existait et la table des groupes de processus est alimentée par script SQL exclusivement.
Ma mission consistait à ajouter un onglet dans un écran existant pour présenter les groupes de processus dans un tableau.
